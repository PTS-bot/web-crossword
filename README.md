# 🧩 Web Crossword Game

เกมอักษรไขว้ออนไลน์ พัฒนาด้วย Nginx + Node.js + MongoDB + Vue.js  
รันบน Docker Compose พร้อมใช้งานทันที

---

## 🚀 วิธีรัน

```bash
# เริ่มต้น Stack ทั้งหมด
docker-compose up --build -d

# ดู logs แบบ real-time
docker-compose logs -f

# หยุด Stack (ข้อมูลยังอยู่)
docker-compose down

# หยุด + ลบ Volume ทั้งหมด (reset ฐานข้อมูล)
docker-compose down -v
```

เมื่อรันแล้วเข้าใช้งานที่: **http://localhost:8080**

## 🎮 หน้าเกม (Player)

- **การเลือกด่าน & ปริมาณคำ**: เลือกหมวดหมู่โจทย์ และเลือกจำนวนคำสุ่ม (ล็อคขั้นต่ำ 3 คำ โดยจะดีดตัวเลื่อนกลับไปที่ 3 คำโดยอัตโนมัติหากพยายามลากต่ำกว่านั้น เพื่อให้สร้างตารางอักษรไขว้ได้)
- **ระบบคำใบ้หลายภาษา (2nd Language Clues)**: 
  - หากด่านนั้นมีการอัปโหลดคำใบ้ภาษาที่ 2 (คอลัมน์ที่ 3 ใน CSV) ผู้เล่นสามารถเปิดใช้งานสวิตช์ความปลอดภัย (Safety Toggle) ในหน้าต่างคำใบ้ จากนั้นสามารถคลิกกล่องคำใบ้ทีละข้อเพื่อสลับภาษาไปมาแบบ 3D ได้
  - หากมีการสลับภาษาอย่างน้อย 1 ข้อ จะมีสัญลักษณ์ **"เกมนี้อยู่ในหมวดหลายภาษา"** แสดงอยู่มุมขวาบนของขอบเมนู เพื่อบ่งบอกว่ากำลังเล่นในโหมดหลายภาษา
- **การควบคุมตาราง**: ลากเมาส์/สัมผัสเพื่อเลื่อนตาราง (Pan) · เลื่อนล้อเมาส์/กางนิ้วเพื่อซูม (Zoom)
- **เครื่องมือช่วยเหลือ**: 
  - กดปุ่ม **คำใบ้ (Clues)** เพื่อดูคำใบ้ทั้งหมด (popup)
  - กดปุ่ม **เฉลย (Reveal)** เพื่อเปิดโหมดเฉลย → ดับเบิลคลิกที่ช่องในตารางเพื่อเปิดตัวอักษร
- **ตารางคะแนน (Leaderboard)**:
  - แสดงความเร็วในการทำเวลา (Words per Second)
  - มีตัวเลือกกล่องตัวกรอง **"2nd Language Clues"** ในแผงลีดเดอร์บอร์ด เพื่อกรองแสดงเฉพาะผู้ที่ผ่านรอบเกมที่มีการกดสลับอ่านคำใบ้ภาษาที่ 2 เท่านั้น (จะมีเครื่องหมาย `(2)` นำหน้าชื่อด่าน)

## 🗂️ ฟีเจอร์ Admin

| ฟีเจอร์ | รายละเอียด |
|--------|-----------|
| **สร้าง Directory** | สร้างหมวดหมู่โจทย์ เช่น `network_lv1`, `vocab_easy` |
| **อัปโหลด CSV** | ลากไฟล์ CSV มาวาง หรือคลิกเพื่อเลือกเพื่อนำคำศัพท์เข้าระบบ |
| **ดูข้อมูล** | ตรวจสอบคำศัพท์และข้อความคำใบ้ทั้งภาษาที่ 1 และ 2 |
| **ลบหมวดหมู่** | ลบพร้อมคำศัพท์ทั้งหมด |
| **แดชบอร์ด** | ดูสถิติการเติบโต และความเร็วเฉลยเทียบกับกลุ่มผู้เล่นคนอื่น |

## 📄 รูปแบบไฟล์ CSV

ไฟล์ CSV สามารถรองรับได้ถึง **3 คอลัมน์** คั่นด้วยจุลภาค (`,`):

```
คำศัพท์,คำใบ้ 1,คำใบ้ 2 (ทางเลือก)
switch,อุปกรณ์เครือข่าย,A device that connects devices in a LAN.
host,เครื่องคอมพิวเตอร์ในเครือข่าย,A computer connected to a network.
server,เครื่องให้บริการ,A computer that provides services.
```

- **คอลัมน์ที่ 1**: คำศัพท์ (ตัวอักษรภาษาอังกฤษ ไม่มีช่องว่าง)
- **คอลัมน์ที่ 2**: คำใบ้หลัก (ภาษาที่ 1)
- **คอลัมน์ที่ 3 (ไม่บังคับ)**: คำใบ้รอง (ภาษาอังกฤษ/ภาษาที่ 2 - สำหรับใช้ในโหมดสลับภาษา)
- รองรับ header row — ถ้าแถวแรกไม่ใช่คำศัพท์จะข้ามไปอัตโนมัติ
- คำที่มีความยาว < 2 ตัวอักษรจะถูกข้ามไป
