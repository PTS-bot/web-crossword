🚀 Vue.js Dockerized Web Template

This is a modern reactive fullstack template powered by Nginx, NodeJS (Express), MongoDB, and Vue.js. All services are fully dockerized with hot-reloading configurations enabled.

Nginx Proxy

Acts as web server serving Vue.js files and acts as reverse proxy, routing /api/* to NodeJS. Listens on host port 8080.

NodeJS Backend

Scalable Express server running inside Node 18 Alpine. Leverages session tokens and Mongoose to interact with MongoDB.

MongoDB NoSQL

High performance document database. Holds user profiles and note collections with persistent host volume bindings.

Docker Management Commands

# Build and spin up the Vue.js stack
docker-compose up --build -d
# Stream logs from all 3 services
docker-compose logs -f
# Shut down stack without losing data
docker-compose down
# Complete database reset
docker-compose down -v
