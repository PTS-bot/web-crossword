ผมอยากให้หน้า admin มีปุ่ม dashboard ของ user 
เมื่อกดแล้ว จะมี list ของ user ทั้งหมดในทางด้านซ้าย ด้านบนสุดของกล่องด้านซ้ายมีปุ่มค้นหาหรือกรอง user เช่นถ้าพิมพ์ 6414* ก็ให้แสดง user ที่ขึ้นด้วยด้วย 6414 ขึ้นไปเท่านั้น 
ด้านขวา เป็น dashboard ของ user เมื่อเรากดเลือก user จะแสดง ค่าสถิติของ user นั้นๆ ว่าเล่นในแต่ละรอบได้คะแนนเท่าไร แกน x เป็นรอบตามวันที่ y เป็น คะแนนที่ได้ให้เป็นกราฟเส้น 
ให้มีเส้นค่าเฉลี่ยของ user ที่เรากรอง 6414* และถ้าเราเลือก user จะมีอีกเส้น ของ user นั้นด้วย
ถ้าจำนวนรอบไม่ได้ก็ให้เส้นหยุดเพียงเท่านั้น เพื่อดูอัตตราการเดิบโตในแต่ละรอบของ user นั้น
